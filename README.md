# HR Analytics Dashboard

## Project Overview
This project analyzes employee data to identify workforce trends, attrition patterns, salary distribution, and employee demographics. The dashboard helps HR teams make informed decisions using data visualization.

## Objectives
- Analyze employee data
- Identify attrition trends
- Calculate HR KPIs
- Visualize insights using charts
- Build an interactive HR dashboard

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Microsoft PowerPoint

## KPIs
- Total Employees
- Active Employees
- Employees Left
- Attrition Rate
- Retention Rate
- Average Salary

## Charts
- Department-wise Employees
- Attrition Pie Chart
- Age Distribution
- Monthly Income Distribution
- Correlation Heatmap

## Conclusion
The HR Analytics Dashboard provides valuable insights into employee data and helps organizations make data-driven HR decisions.
